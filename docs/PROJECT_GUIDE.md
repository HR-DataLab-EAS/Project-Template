# Project Guide

## Overview
This is a data science project template that provides a structured approach to data analysis and processing.

## Key Features
- **Modular structure**: Organized code for data loading, processing, and analysis
- **Sample data**: Includes dummy data for testing and demonstration
- **Documentation**: Comprehensive guides for setup and usage
- **Best practices**: Follows Python data science conventions

## Quick Start

1. **Load sample data:**
   - Sample data is available in `Data/raw/sample_data.csv`
   - Contains 12 records with customer information

2. **Process data:**
   - Use functions in `src/data.py` to load and clean data
   - Processed data is saved to `Data/processed/`

3. **Analyze results:**
   - Use functions in `src/analysis.py` for statistical analysis
   - Generate correlation matrices and summary statistics

## Data Schema

The sample dataset includes:
- `id`: Unique identifier
- `name`: Customer name
- `age`: Customer age
- `income`: Annual income in EUR
- `city`: Location in Netherlands
- `registration_date`: Date customer registered

## Next Steps

1. Modify the sample data according to your needs
2. Extend the analysis functions in `src/analysis.py`
3. Add more processing steps to `src/data.py`
4. Update documentation as needed
